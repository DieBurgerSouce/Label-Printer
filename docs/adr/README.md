# Architecture Decision Records (ADR)

This directory contains Architecture Decision Records for Screenshot_Algo.

## What is an ADR?

An Architecture Decision Record (ADR) is a document that captures an important architectural decision made along with its context and consequences.

## Creating a New ADR

1. Copy `000-template.md` to a new file with the next number
2. Fill in all sections
3. Set status to "Proposed"
4. Submit as PR for team review
5. Update status to "Accepted" when approved
6. Update this README's index

## ADR Statuses

- **Proposed**: Under discussion
- **Accepted**: Decision has been made
- **Deprecated**: No longer valid
- **Superseded**: Replaced by another ADR
